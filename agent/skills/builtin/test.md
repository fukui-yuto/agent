このプロジェクトのテストを実行して結果を報告してください。

手順:
1. プロジェクトのテストフレームワークを確認する（pytest / unittest / jest / go test 等）
2. テストコマンドを実行する（`pytest` / `npm test` / `go test ./...` 等）
3. 失敗したテストがある場合は原因を分析する
4. テストの結果サマリーを報告する

テスト設定ファイル（pytest.ini / package.json 等）があれば参照してください。
